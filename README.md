node


## Getting Started

```
git clone https://github.com/nayfin/tour-of-nativescript.git

cd tour-of-nativescript

npm i

git checkout 01-initial-clone-of-heroes

ng serve
```

You should see the classic Tour of Heroes app spin up on `localhost:4200`.
